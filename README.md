# CahierDeTexte
Ci un cahier de texte numerique pour un etablissement scolaire ESITEC fait en langage JAVA

tu clone d'abord le projet 
clone  gh repo clone Benimian/CahierDeTexte
Man avant toutes choses active mysql et apacheweb ensuite cree une base de donnee avec le nom cahier_texte ne met pas de majiscule et autres ensuite importe le fichier sql que je vais envoyer par whatssap
des que tu deplace dans le projet avec la commande "cd" tape ce qui est entre "" pour executer : " mvn compile && mvn exec:java -Dexec.mainClass="com.esitec.cahier.Main"  "
